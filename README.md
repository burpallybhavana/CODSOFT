# Rock Paper Scissors Console Based Game

# Project Overview
This project is a Python-based console application that simulates the classic Rock–Paper–Scissors game.  
The user plays against the computer, which makes random choices. The game runs for a fixed number of rounds and displays the final winner based on the scores.



#  Features
- User vs Computer gameplay
- Random computer choice using Python’s 'random' module
- Fixed number of rounds
- Score tracking for both user and computer
- Exit option to stop the game early
- Final result and winner announcement



# Technologies Used
- Python 3
- 'random' module



#  How to Run the Program
1. Open Command Prompt / Terminal  
2. Navigate to the project directory  
3. Run the following command:
   bash
   python rock_paper_scissors.py
